Let's talk about html!

Hypertext Markup Language is the standart markup language for documents designed to be displayed in a web browser. It can be assisted by technologies such as Css and scripting languages such as JavaScript

Web browsers receive HTML documents from a web server or from local storage and render the documents into multimedia web pages. HTML describes the structure of a web page semantically and originally included cues for the appearance of the document.

Let's talk about css!

Cascading Style Sheets is a style sheet language used for describing the prsentation of a document written in a markup language like HTML. CSS is a cornerstone technology of the World Wide Web, alongside HTML JavaScript.


CSS is designed to enable the separation of presentation and content, including layout, colors, and fonts. This seperation can improve content accesibility, provide more flexibility and control in the specification of presentation chararacteristics, enable multiple web pages to share formatting by specifying the relevant CSS in a seperate .css file which reduces complexity and repetition in the structural content as well as enabling the .css file to be cached to improve the page load speed between the pages that share the file and its formatting.
